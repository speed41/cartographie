Aide pour le dossier infos par secteur


Ce dossier est destiné à être renseigné par les secteurs eux même pour maintenir les informations de leurs moyens à jour de manière simple.
Ce dossier devra donc être accécible par le serveur et par les systèmes (ou personnes) qui vont les maintenir à jour.


Dans ce dossier, chaque secteur peut y avoir un dossier à son nom. Dans chaque dossier 2 types de fichiers :

Exemple, nom du moyen : "riboucleuse"

Informations du moyen

le nom du fichier :
"infos_"nom du moyen.txt (infos_riboucleuse.txt)

Ces informations pourront êtres affichées sur le moyen lui même. Elles permettent, par exemple d'affichier le n° d'immobilisation, sa fonction son numéro d'étalonnage ...

Etat du moyen

le nom du fichier :
nom du moyen.txt (riboucleuse.txt)

Ces informations seront affichées sous le moyen. Elles permettent d'afficher l'état d'avancement du test en cours grace à une structure simple.

Si une ligne commence par :
titre=
Le texte après sera écrit en premier avec une taille supérieur au reste du texte
couleur=
indique la couleur en rgb (#FF00FF) de la zone
progresse=
affiche une progresse barre, la valeur à indiquée est en % (sans le symbol %)
bulle=
Le texte qui sera dans l'info bulle.
Le texte sera affiché de manièe simple
Par contre si le contenu du fichier texte est sous format html, il sera interprété comme tel, à condition que la première ligne soit : <HTML>
